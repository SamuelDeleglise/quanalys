## Install

`pip install labmate`

## Installation in dev mode

`pip install -e .[dev]`

`python setup.py develop`

## Usage

### SyncData

SyncData is a way to open h5 file and change it.

```python

```

`docs/examples/acquisition_and_analysis_notebook.ipynb`
